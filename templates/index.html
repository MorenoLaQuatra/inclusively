This is a demo for the Inclusively project. It aims at developing deep learning models that can detect and correct non-inclusive language.

It leverages an NLP pipeline that includes:

  • Detection: Detecting non-inclusive language in a text.
  • Rewriting: If some non-inclusive expression is detected, the model will rewrite the text to make it inclusive.
